### Subroutine addresses for ALS and Dyna 8085 kits:
#### (won't work with the simulator)


#### For Dyna Kit
* RDKBD : 0x02E7

* MODIDT: 0x036E

* MODIAD: 0x0362

#### For ALS Kit
* RDKBD: 0x0634

* UPDDT: 0x06D3

* UPDAD: 0x06BC
